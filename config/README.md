# Config

YAML config files for scrapers and other project settings.

- **scrapers.yaml** – Scraper settings (e.g. `nudie_jeans`: base_url, locale_path, currency, mens_fits with name/slug). Add other brands here as needed.

Currency for Nudie Jeans is set in `nudie_jeans.currency` (e.g. `CAD`). Prices are stored as numbers in CSV; the currency label is for reference and downstream use.
